# Movie Registration App

## 📌 About the Project
This project is a movie registration application built with JavaScript. Users can fill out a form with movie information, save the data in localStorage, and view the registered movies on a separate page.

## 🚀 Features
- Movie registration form
- Real-time validation
- Multiple input types (text, date, radio, checkbox and select)
- Data persistence with localStorage
- Separate page to display saved movies
- Delete registered movies

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- localStorage

## 🎯 Purpose
This project was created to practice DOM manipulation, form handling, validation, browser storage, and dynamic rendering of saved data.
