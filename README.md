# Movie Registration App

## 📌 About the Project
This project is a movie registration application built with JavaScript. It allows users to register movie information through an interactive form with validation and data persistence using localStorage.

## 🚀 Features
- Real-time form validation
- Multiple input types (text, date, radio, checkbox and select)
- Movie data registration
- Data persistence with localStorage
- User interaction through a dynamic form interface

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- localStorage

## 🎯 Purpose
This project was created to practice JavaScript form handling, validation, DOM manipulation and browser storage.
